# Melbourne House Price Prediction

## Project Overview
Predicting Melbourne house prices using Random Forest ML model.

## Dataset
- Source: Kaggle Melbourne Housing Dataset
- Size: 13,580 houses with 21 features

## Steps
1. Data Loading and Exploration
2. Data Cleaning (handling missing values)
3. Exploratory Data Analysis
4. Machine Learning Model (Random Forest)
5. Model Evaluation

## Results
- Model Score: 0.66
- Mean Absolute Error: $238,457

## Key Findings
- Distance from CBD is most important feature
- More rooms = higher price
- Location strongly affects price
